## 🚀 About

MechForge is my **100-Day Challenge** to become a mechanical engineer capable of developing professional engineering software using Python.

Rather than learning programming through isolated exercises, I am building real engineering tools that solve practical mechanical engineering problems.

This repository documents my journey from Python beginner to engineering software developer.

---

## 🎯 Objectives

- Learn Python through real engineering projects
- Develop professional software engineering practices
- Build a strong GitHub portfolio
- Apply AI to mechanical engineering
- Prepare for internships, master's scholarships, and engineering careers
- Create a complete Mechanical Engineering Toolkit
